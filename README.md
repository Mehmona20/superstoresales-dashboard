# Superstore Sales Dashboard

## Project Overview

This project was created using Power BI to analyze Superstore sales data. The dashboard shows sales trends, category performance, profit analysis, and regional sales performance.

## Tools Used

- Power BI
- Power Query
## Dataset
- Source: Kaggle
- Dataset Name: Sample Superstore Dataset
- Format: CSV
- Records: 9,994
- Columns: 21

## Data Cleaning
- Imported the dataset into Power BI Power Query.
- Promoted the first row as headers.
- Changed Order Date and Ship Date to Date format.
- Changed Sales, Profit, and Discount to Decimal Number.
- Changed Quantity to Whole Number.
- Verified the Postal Code data type.
- Checked for missing values (no significant missing values found).
- Checked for duplicate records (no duplicate records found).
- Verified the dataset before creating visualizations.

## Dashboard Features
- Monthly Sales Trend
- Sales by Category
- Profit vs Sales by Category
- Regional Sales Analysis
- KPI Cards (Total Sales, Total Profit, Total Orders)

## Business Insights
- Sales increased significantly over the years.
- Technology generated the highest sales.
- West region recorded the highest sales.
- Technology generated much higher sales than Office Supplies, but both categories earned similar profit.
- Furniture generated lower profit despite similar sales.
